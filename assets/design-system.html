Theming & Elevation
Toggle dark mode with the moon/sun button in the top bar (or call AG.toggleTheme()). Theme persists in localStorage and respects the OS preference on first visit. Every component reads design tokens, so the whole UI flips with zero per-component changes — mirroring Flutter's ThemeData.dark().
